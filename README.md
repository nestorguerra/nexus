# Nexus Governance Partners - GitHub Pages

Web estatica lista para publicar en GitHub Pages.

## Como subirlo

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos de esta carpeta, no la carpeta contenedora.
3. En GitHub, entra en `Settings > Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona la rama `main` y la carpeta `/root`.
6. Guarda y espera a que GitHub publique la web.

## Archivos importantes

- `index.html`: pagina principal.
- `assets/images/`: imagenes de la presentacion.
- `assets/videos/`: video embebido en la slide de Tendencia 7.
- `assets/fonts/`: fuentes usadas por la presentacion.
- `.nojekyll`: evita que GitHub procese la web con Jekyll.

No necesita compilacion, instalacion de dependencias ni servicios externos.
